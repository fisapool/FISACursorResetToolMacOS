Setup Instructions for Cursor AI

Follow these steps to set up and run the Cursor AI scripts for the first time on your macOS system.

Step 1: Open Terminal

Open the Terminal application on your macOS using one of these methods:

  • Navigate to Applications > Utilities > Terminal
  • Use Spotlight Search (‚åò + Space) and type "Terminal"

Step 2: Navigate to the Script Directory

You'll need to navigate to where your scripts are located. Here's how:

Finding Your Script's Location:

  1. In Finder, locate your script file
  2. Right-click on the file and select "Get Info" (or use ‚åò + I)
  3. Look under the "Where:" section to find the full path

Then use this command in Terminal, replacing the path with your actual script location:

cd /path/to/directory

Tip: You can also drag and drop the folder from Finder into Terminal after typing 'cd ' to automatically insert the path.

Step 3: Make Scripts Executable

Before running the scripts, you need to make them executable. This is a one-time setup:

chmod +x cursor-terminal-script
chmod +x launcher

Note: The chmod +x command gives the script permission to execute. You only need to do this once for each script.

Step 4: Run the Launcher Script

Execute the launcher script to start the process. This will open a new Terminal window and run the cursor-terminal-script:

./launcher

Note: If prompted, enter your Mac password. This is your user account password used to log in to your Mac.

Step 5: Follow On-Screen Instructions

Once the script is running, follow the on-screen instructions to complete the setup and installation process.
